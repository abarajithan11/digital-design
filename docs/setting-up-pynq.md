# Setting up PYNQ

## Prerequisites

- PYNQ-supported board
- USB cable and power supply
- Access to the board image and documentation

## Steps

1. Download the correct PYNQ image for your board.
2. Flash the image to an SD card.
3. Boot the board with the SD card inserted.
4. Connect to the board over network/USB.
5. Open Jupyter and verify the environment.

## Validation

- Board boots correctly
- Jupyter is reachable
- A sample notebook runs successfully
