# Setting up Docker

## Prerequisites

- Docker Desktop installed
- Virtualization enabled in BIOS/UEFI

## Steps

1. Install Docker Desktop.
2. Start Docker Desktop and verify it is running.
3. Pull a base image for the course work.
4. Run a test container.

## Quick Check

Use:

```bash
docker --version
docker run --rm hello-world
```

If both commands succeed, your setup is ready.
